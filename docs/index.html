Overview

What we found

Across 15 European countries, discussions about sustainable coffee are overwhelmingly positive — driven by appreciation for ethical sourcing, fair trade certification, and eco-friendly packaging. Scepticism surfaces mainly around carbon neutrality claims and consumer pricing.

Reproduce this analysis

Run it yourself in 5 steps

The full pipeline is open source. You need R ≥ 4.2 and a free X Developer account.

1

Clone the repository

Download the project from GitHub to your machine.

git clone https://github.com/Anesu-Chimbi/Sustainable-coffee-perceptions-europe
2

Get your Bearer Token

Create a free X Developer account at developer.twitter.com, create an App, and copy your Bearer Token from Keys & Tokens.

3

Set up credentials

Create a .env file from the template and paste in your token.

cp .env.example .env
4

Run the pipeline

Open R or RStudio in the project folder and run the master script.

source("Sustainable-coffee/scripts/run_all.R")
5

View your results

Charts and maps appear in Sustainable-coffee/outputs/. Open this page locally or push to GitHub Pages.

🔗

Data collection

X API v2 recent search. Keywords: "sustainable coffee", "eco coffee", "fair trade coffee". Auth: Bearer Token via httr2. Falls back to mock data if no token is set.

💬

Sentiment analysis

Bing lexicon via tidytext. Word scores aggregated per tweet into a normalised compound score. Threshold: positive > 0.05, negative < –0.05.

🗺️

Geospatial mapping

Country boundaries from rnaturalearthdata (50m). Choropleth and bubble maps rendered with ggplot2 + sf + patchwork.
